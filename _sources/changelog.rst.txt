Changelog
=========

All notable changes to this project will be documented in this file.

Version 0.1.0 (2026-01-05)
--------------------------

Initial Release
~~~~~~~~~~~~~~~

Features
^^^^^^^^

* Task management with CRUD operations
* Priority levels: LOW, MEDIUM, HIGH, CRITICAL
* Task statuses: TODO, IN_PROGRESS, COMPLETED, CANCELLED
* Due date tracking with overdue detection
* Persistent JSON storage
* Task filtering by status, priority, and overdue status
* Task search by title and description
* Full type hints and type safety
* Comprehensive test suite with >80% coverage

Development
^^^^^^^^^^^

* Python 3.9+ support
* Matrix CI testing on Python 3.9, 3.10, 3.11
* Code quality: Black, Flake8, Pylint, MyPy
* Security scanning with Bandit
* Automated releases with changelog generation
* Documentation with Sphinx and GitHub Pages
* Complete CI/CD pipeline with GitHub Actions
